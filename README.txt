README

Este programa muestra a un lindo pero molesto perro en la consola de tu computador, lo unico
que tienes que hacer es invocarlo y este aparecera por horas sin parar, ya sea por si quieres dejarlo de fondo
o simplemente dejar una marca de que estuviste alli

para ejecutar el programa es necesario compilar primero el archivo "EndlessDogoCicle.cpp" ya sea de la siguiente manera
g++ EndlessDogoCicle.cpp -o <nombreArchivoEjecutable>

o de la manera mas simple, solo ejecuta el archivo "dogoIn" en la consola de la siguiente forma
./dogoIn
